# Forest Fire Weather Index Prediction Web App

This project predicts the Forest Fire Weather Index (FWI) using meteorological inputs.

Users enter environmental parameters such as temperature, humidity, wind speed, and rainfall, and the system predicts the fire risk using a trained machine learning model.

## Features

- Machine learning model using Ridge Regression
- Feature scaling using StandardScaler
- Flask web application interface
- Real-time prediction from user inputs

## Tech Stack

Python  
Flask  
Scikit-learn  
Pandas  
NumPy  
HTML

## Project Structure

application.py → Flask backend  
templates/index.html → Web interface  
ridge.pkl → trained ML model  
scaler.pkl → feature scaler
